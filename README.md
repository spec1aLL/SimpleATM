# SimpleATM

Консольное приложение на языке C#, реализующее базовую логику работы банкомата (ATM). Проект демонстрирует применение фундаментальных концепций программирования: работа с коллекциями, циклами и условной логикой.

## Функциональные возможности

Система предоставляет следующий набор операций:
* **Авторизация**: имитация проверки PIN-кода для доступа к счету.
* **Баланс**: запрос текущего остатка денежных средств.
* **Депозит**: пополнение счета с обновлением данных в реальном времени.
* **Снятие**: вывод средств с проверкой достаточности лимита.
* **Завершение сессии**: безопасный выход из системы.

## Технологический стек

* **Runtime**: .NET 8.0 / 9.0 (C# 12/13)
* **Paradigm**: Procedural / Object-Oriented Programming
* **CLI**: Стандартный поток ввода-вывода (System.Console)

## Требования

Для сборки и запуска приложения необходимы:
* .NET SDK (версии 8.0 или выше)
* Git

## Установка и запуск

### Способ 1: Через IDE (рекомендуется)

1. Проверьте версию установленного SDK:
    ```bash
    dotnet --version
    ```

2. Клонируйте репозиторий:
    ```bash
    git clone github.com
    cd SimpleATM
    ```

3. Откройте папку в **Visual Studio**, **JetBrains Rider** или **VS Code**.
4. Запустите отладку клавишей `F5`.

### Способ 2: Через терминал

1. Выполните клонирование и переход в директорию:
    ```bash
    git clone github.com
    cd SimpleATM
    ```

2. Выполните команду запуска:
    ```bash
    dotnet run
    ```
    _Примечание: Если в директории несколько проектов, используйте:_  
    `dotnet run --project SimpleATM.csproj`
